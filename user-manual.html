User Manual
Overview
The website provides a user interface to make submissions for optimisation problems. Each entity of the contest and their descriptions are given below.
Passcode
Passcode is an entity which uniquely identifies each team. Teams must enter their passcode into the given textbox and verify their passcode. Once the verification is done, the user is acknowledged with their teamname.
Code File
The code file must be uploaded for each submission. This is to ensure that the participants have genuinely solved the problem by writing code. Submissions without a code file will not be accepted.
Output Files
Output files must be uploaded to evaluate the score for each test case. It is not compulsory to upload the output file for each test case. However, it is compulsory to upload at least one output file for each test case.
Submission
Once the user uploads the code file and clicks on submit, the submission undergoes evaluation. The result of the submission is displayed in the "My Submissions" page. The result will appear after a few minutes after the submission is made.
Leaderboard
The leaderboard contains a list of teams ranked according to their scores. If the scores of two teams are equal, then they are ranked according to the time taken to attain the score.
